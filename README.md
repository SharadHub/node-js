# Backend Development — Node.js & Express
---

## What This Covers

End-to-end backend development — from core Node.js concepts to deploying a live API
with authentication and a real database.

---

## Skills & Topics

**Node.js Fundamentals**
- How Node.js works under the hood (V8 engine, event loop, non-blocking I/O)
- Module systems, NPM, and dependency management

**Asynchronous Programming**
- Callbacks, Promises, and async/await
- File system operations, event emitters, and streams

**Web Servers & Express.js**
- Building HTTP servers from scratch
- Routing, middleware, and request/response handling
- MVC pattern for clean project structure

**REST API Development**
- Designing and building full CRUD APIs
- Input validation, error handling, and route organization

**MongoDB & Mongoose**
- NoSQL database design — schemas, models, and queries
- Data relationships: embedding vs. referencing

**Authentication & Security**
- Password hashing with bcrypt
- JWT-based authentication and protected routes
- CORS configuration

**Deployment**
- Environment variables and production configuration
- Deploying Node.js apps to cloud platforms (Render / Heroku)
